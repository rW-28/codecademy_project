# codecademy_project
Fotomatic – Responsive Photography Website

A responsive photography landing page built with **HTML, CSS, and JavaScript** during the **Codecademy Front-End course**.  
The goal of this project was to practice adaptive web design and implement a visually appealing layout that works seamlessly on desktop, tablet, and mobile screens.

Features
- Fully responsive layout for all devices  
- Modern and minimalist photography theme  
- Interactive navigation and dynamic sections  
- Designed with accessibility and readability in mind  
- Developed as part of Codecademy’s front-end curriculum

Technologies Used
- **HTML5** – semantic structure  
- **CSS3** – media queries, flexbox, responsive design  
- **JavaScript (ES6)** – interactivity and dynamic content
